```math
\begin{cases} \log_{e}(x) = z \cdot 1 \leftarrow e^{z \cdot 1} = e^{z^{1}} = x\\
\log_{e}(x^{y}) = z \cdot y \leftarrow e^{z \cdot y} = e^{z^{y}} = x^{y} \end{cases}
```
```math
\downarrow
```
```math
\log_{e}(x^{y}) = y(\log_{e}(x))
```

```math
\begin{cases} m = \log_{e}(x)\\
n = \log_{e}(y) \end{cases}
```
```math
\begin{cases} e^{m} = x\\
e^{n} = y \end{cases}
```
```math
\frac{x}{y} = \frac{e^{m}}{e^{n}} = e^{m - n}
```
```math
\downarrow
```
```math
\log_{e}(x / y) = m - n = \log_{e}(x) - \log_{e}(y)
```
